# Projekt áttekintése

A projekt egy Twitter/X-ihlette közösségi média webalkalmazás, amelyen a felhasználók bejegyzéseket (posztokat) hozhatnak létre, követhetik egymást, interakcióba léphetnek a tartalmakkal (like, reposzt, hozzászólás), és értesítéseket kapnak mások tevékenységéről. Az alkalmazás tartalmaz adminisztrátori funkciót is, amellyel a moderátorok felfüggeszthetik a visszaélő felhasználókat.

Az alkalmazás kliens-szerver architektúrában épül fel: a backend REST API-t biztosít, a frontend pedig egy önálló, böngészőben futó Single Page Application (SPA).
